



 # User Management Secure API Platform

A secure and scalable backend application built using Node.js, Express, MongoDB, and JWT authentication.

## Features
- User Registration & Login
- JWT-based Authentication
- Protected Routes
- Role-based Access Control (Admin/User)
- Secure Password Hashing (bcrypt)
- RESTful API Architecture

## Tech Stack
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (JSON Web Tokens)
- bcrypt
- dotenv

## API Endpoints
- POST /api/auth/register
- POST /api/auth/login
- GET /api/user/profile (Protected)

## Security
- Password hashing
- Token-based authentication
- Middleware-protected routes

## Future Enhancements
- Refresh Tokens
- Email Verification
- Forgot Password
- Rate Limiting
- Admin Dashboard
